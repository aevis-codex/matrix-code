# 第 87 阶段计划：本地执行工作区与摘要成员权限

## 目标

将本地执行代理中的工作区授权创建、执行摘要读取和任务日志读取接入 Sa-Token 登录态与项目成员权限，避免未登录用户或非项目成员注册本地根目录、查看执行摘要或读取命令输出。

## 成功标准

- 未携带登录身份访问 `/workspaces`、`/summary`、`/commands/{taskId}/logs` 返回 401。
- 非当前项目成员访问上述接口返回 403。
- 当前项目 ACTIVE 成员可以授权工作区、读取执行摘要和查看任务日志。
- 桌面端调用上述接口时可携带当前登录用户身份头和 actor token，同时兼容旧 serverUrl 调用。
- 后端定向测试、前端定向测试、全量回归、真实运行环境检查和敏感信息扫描通过。
